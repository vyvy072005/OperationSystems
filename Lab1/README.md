# README:  Командная строка UNIX

Этот документ описывает использование моей собственной командной строки UNIX, написанной на языке C. Командная строка предоставляет базовый набор команд.

## Доступные команды

| Команда        | Описание                                     
|----------------|-----------------------------------------------|
| cd [каталог]   | Изменяет текущую директорию.                  | 
| clr            | Очищает экран.                                | 
| dir [каталог]  | Выводит содержимое указанного каталога.       |
| environ        | Выводит список переменных окружения.          |
| echo [текст]   | Выводит указанный текст на экран.             | 
| help           | Выводит список доступных команд и их описание.| 
| pause          | Приостанавливает выполнение до нажатия Enter. | 
| quit           | Завершает работу командной строки.            |


## Перенаправление ввода/вывода

Командная строка поддерживает перенаправление ввода и вывода с помощью операторов:

> : Перенаправляет стандартный вывод (stdout) в указанный файл. Файл будет перезаписан.
>>: Перенаправляет стандартный вывод (stdout) в указанный файл, добавляя данные в конец файла.
< : Перенаправляет стандартный ввод (stdin) из указанного файла.


*Примеры:*

*   echo "Привет, мир!" > output.txt - Записывает "Привет, мир!" в файл output.txt.
*   dir >> log.txt - Добавляет содержимое текущей директории в конец файла log.txt.
*   wc < input.txt - Считает слова в файле input.txt.

## Фоновые процессы

Для запуска команды в фоновом режиме, добавьте символ `&` в конце команды. Команда будет запущена в отдельном процессе, и командная строка не будет ждать её завершения.

*Пример:*

sleep 10 & - Запускает команду `sleep 10` в фоновом режиме.

## Обработка ошибок

Командная строка обрабатывает ошибки ввода-вывода и другие ошибки, выводит сообщения об ошибках с использованием perror().

## Компиляция

Для компиляции кода необходимо использовать компилятор GCC
